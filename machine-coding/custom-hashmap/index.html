    43. Design a Custom HashMap

    üìú Problem Statement

    Design an in-memory Custom HashMap that stores String keys and String values.

    You must implement:

    • Custom hashing
    • Buckets
    • Collision handling
    • Dynamic rehashing (grow & shrink)

    ‚ùó Important: Do NOT use any built-in Map, Set, or Dictionary for storing key‚Äìvalue pairs.

    üß† Hashing & Buckets

    Hash Function

    hash(key) = (key.length() ^ 2) + sum(charValue)

    Where:

    • a = 1, b = 2, ‚Ä¶ z = 26

    Example:

    key = "abcd"
hash = 4*4 + (1+2+3+4) = 26

    Bucket Index

    bucketIndex = hash(key) % bucketsCount
    • Initial bucketsCount = 2
    • Collisions go into the same bucket (linked list)

    üìä Load Factor & Rehashing

    Load Factor

    LoadFactor = round2(size / bucketsCount)
round2(x) = Math.round(x * 100.0) / 100.0
    • minLoadFactor < maxLoadFactor
    • All load factor values are rounded to 2 decimals

    Rehash Rules

    After every put() and remove():

    üîº Grow

    • If LoadFactor > maxLoadFactor
    • Double buckets: 2 ‚Üí 4 ‚Üí 8 ‚Üí ‚Ä¶

    üîΩ Shrink

    • If LoadFactor < minLoadFactor
    • Halve buckets until valid
    • ‚ùó Never shrink below 2 buckets

    Rehashing means:

    • Create new buckets
    • Reinsert all existing entries using:
    newBucketIndex = hash(key) % newBucketsCount

    üß© API Design

    Constructor

    CustomHashMap(double minLoadFactor, double maxLoadFactor)

    put

    void put(String key, String value)
    • key, value length: 1..20
    • key contains only a-z
    • Update value if key exists
    • Size increases only for new keys

    get

    String get(String key)
    • Returns value if found
    • Returns "" if not found

    remove

    String remove(String key)
    • Removes entry
    • Returns removed value
    • Returns "" if key not found

    getBucketKeys

    List<String> getBucketKeys(int bucketIndex)
    • Invalid index ‚Üí empty list
    • Returns keys in that bucket
    • Sorted lexicographically ascending

    size

    int size()

    bucketsCount

    int bucketsCount()

    üß™ Examples

    Example 1 ‚Äì Basic Operations

    CustomHashMap map = new CustomHashMap(0.25, 0.75);

map.put("a", "one");
map.put("bb", "two");

map.get("a")      // "one"
map.get("x")      // ""

map.put("a", "ONE");
map.get("a")      // "ONE"

map.remove("bb")  // "two"
map.remove("bb")  // ""
map.size()        // 1

    Example 2 ‚Äì Collision Handling

    // bucketsCount = 2
map.put("a", "v1");   // hash=2 ‚Üí bucket 0
map.put("c", "v2");   // hash=4 ‚Üí bucket 0

map.getBucketKeys(0)  // ["a", "c"]
map.getBucketKeys(1)  // []

    Example 3 ‚Äì Rehash Grow

    map.put("a", "1");    // LF=0.50
map.put("b", "2");    // LF=1.00 ‚Üí rehash to 4 buckets

map.bucketsCount()   // 4
map.size()           // 2

    Example 4 ‚Äì Rehash Shrink

    // Suppose size=3, buckets=8
// LF=0.38 < minLoadFactor(0.50) ‚Üí shrink to 4

    üèóÔ∏è Design Overview

    Data Structures Used

    • Array of buckets
    • Each bucket = Singly Linked List
    • Manual lexicographic sorting (no Collections.sort)

    üß† Design Patterns & Concepts

    • Hashing
    • Separate Chaining (Collision Handling)
    • Dynamic Resizing
    • Load Factor Control
    • Manual Rehashing

    üíª Java Implementation

    import java.util.*;

public class CustomHashMap {

    class Entry {
        String key;
        String value;
        Entry next;

        Entry(String key, String value) {
            this.key = key;
            this.value = value;
        }
    }

    private Entry[] buckets;
    private int size;
    private double minLoadFactor;
    private double maxLoadFactor;

    public CustomHashMap(double minLoadFactor, double maxLoadFactor) {
        this.minLoadFactor = round2(minLoadFactor);
        this.maxLoadFactor = round2(maxLoadFactor);
        this.buckets = new Entry[2];
        this.size = 0;
    }

    private double round2(double val) {
        return Math.round(val * 100.0) / 100.0;
    }

    public void put(String key, String value) {
        int index = bucketIndex(key, buckets.length);
        Entry curr = buckets[index];

        while (curr != null) {
            if (curr.key.equals(key)) {
                curr.value = value;
                return;
            }
            curr = curr.next;
        }

        Entry newEntry = new Entry(key, value);
        newEntry.next = buckets[index];
        buckets[index] = newEntry;
        size++;

        rehashIfNeeded();
    }

    public String get(String key) {
        int index = bucketIndex(key, buckets.length);
        Entry curr = buckets[index];

        while (curr != null) {
            if (curr.key.equals(key)) {
                return curr.value;
            }
            curr = curr.next;
        }
        return "";
    }

    public String remove(String key) {
        int index = bucketIndex(key, buckets.length);
        Entry curr = buckets[index];
        Entry prev = null;

        while (curr != null) {
            if (curr.key.equals(key)) {
                if (prev == null) {
                    buckets[index] = curr.next;
                } else {
                    prev.next = curr.next;
                }
                size--;
                rehashIfNeeded();
                return curr.value;
            }
            prev = curr;
            curr = curr.next;
        }
        return "";
    }

    public List<String> getBucketKeys(int bucketIndex) {
        List<String> result = new ArrayList<>();
        if (bucketIndex < 0 || bucketIndex >= buckets.length) return result;

        Entry curr = buckets[bucketIndex];
        while (curr != null) {
            result.add(curr.key);
            curr = curr.next;
        }

        // manual sort
        for (int i = 0; i < result.size(); i++) {
            for (int j = i + 1; j < result.size(); j++) {
                if (result.get(i).compareTo(result.get(j)) > 0) {
                    String tmp = result.get(i);
                    result.set(i, result.get(j));
                    result.set(j, tmp);
                }
            }
        }
        return result;
    }

    public int size() {
        return size;
    }

    public int bucketsCount() {
        return buckets.length;
    }

    private int bucketIndex(String key, int count) {
        return hash(key) % count;
    }

    private int hash(String key) {
        int sum = 0;
        for (char c : key.toCharArray()) {
            sum += (c - 'a' + 1);
        }
        int len = key.length();
        return len * len + sum;
    }

    private void rehashIfNeeded() {
        double lf = loadFactor();

        while (lf > maxLoadFactor) {
            rehash(buckets.length * 2);
            lf = loadFactor();
        }

        while (lf < minLoadFactor && buckets.length > 2) {
            rehash(buckets.length / 2);
            lf = loadFactor();
        }
    }

    private double loadFactor() {
        return round2((double) size / buckets.length);
    }

    private void rehash(int newCount) {
        Entry[] old = buckets;
        buckets = new Entry[newCount];
        size = 0;

        for (Entry head : old) {
            Entry curr = head;
            while (curr != null) {
                insertWithoutRehash(curr.key, curr.value);
                curr = curr.next;
            }
        }
    }

    private void insertWithoutRehash(String key, String value) {
        int index = bucketIndex(key, buckets.length);
        Entry e = new Entry(key, value);
        e.next = buckets[index];
        buckets[index] = e;
        size++;
    }
}

    ‚è±Ô∏è Time & Space Complexity

    Operation Average Worst
    put O(1) O(n)
    get O(1) O(n)
    remove O(1) O(n)
    rehash O(n) O(n)

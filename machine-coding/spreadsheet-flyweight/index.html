    25. Design a Spreadsheet (Excel-like) ‚Äî Flyweight Optimized

    üìú Problem Statement (Recap)

    We are building an in-memory spreadsheet where:

    • Each cell stores text + style

    • Styles include:

      • font name
      • font size
      • bold
      • italic

      • Spreadsheet supports:

      • Inserting rows

      • Inserting columns
      • Setting & getting cell values
      • Single-threaded environment

    ‚ùó Problem With Naive Design

    In a large spreadsheet:

    • Millions of cells
    • Thousands of cells share the same style

    ‚ùå Naive approach creates duplicate style objects ‚û°Ô∏è Massive memory waste

    ‚úÖ Flyweight Pattern Solution

    Core Idea

    Separate intrinsic (shared) state from extrinsic (per-cell) state

    State Type Stored Where
    Style (font, size, bold, italic) Shared Flyweight
    Text + position Individual Cell

    üé® Style Flyweight (Shared)

    class StyleFlyweight {
    final String fontName;
    final int fontSize;
    final boolean isBold;
    final boolean isItalic;

    StyleFlyweight(String fontName, int fontSize,
                   boolean isBold, boolean isItalic) {
        this.fontName = fontName;
        this.fontSize = fontSize;
        this.isBold = isBold;
        this.isItalic = isItalic;
    }
}

    üè≠ Flyweight Factory

    class StyleFactory {

    private static final Map<String, StyleFlyweight> cache = new HashMap<>();

    public static StyleFlyweight getStyle(String fontName, int fontSize,
                                          boolean isBold, boolean isItalic) {

        String key = fontName + "|" + fontSize + "|" + isBold + "|" + isItalic;

        if (!cache.containsKey(key)) {
            cache.put(key, new StyleFlyweight(
                    fontName, fontSize, isBold, isItalic
            ));
        }
        return cache.get(key);
    }
}

    ‚úî Ensures only one instance per unique style

    üß© Cell (Extrinsic State Only)

    class Cell {
    String text;
    StyleFlyweight style;

    Cell(String text, StyleFlyweight style) {
        this.text = text;
        this.style = style;
    }
}

    üèóÔ∏è Spreadsheet Implementation (Flyweight)

    import java.util.*;

public class Spreadsheet {

    private List<List<Cell>> sheet;
    private int rows;
    private int cols;

    // ---------- Initialization ----------
    public Spreadsheet() {
        rows = 5;
        cols = 5;
        sheet = new ArrayList<>();

        for (int i = 0; i < rows; i++) {
            sheet.add(new ArrayList<>(Collections.nCopies(cols, null)));
        }
    }

    // ---------- Add Row ----------
    public void addRow(int index) {
        if (index < 0 || index > rows) return;

        sheet.add(index, new ArrayList<>(Collections.nCopies(cols, null)));
        rows++;
    }

    // ---------- Add Column ----------
    public void addColumn(int index) {
        if (index < 0 || index > cols) return;

        for (List<Cell> row : sheet) {
            row.add(index, null);
        }
        cols++;
    }

    // ---------- Add / Replace Entry ----------
    public void addEntry(int row, int col, String text,
                         String fontName, int fontSize,
                         boolean isBold, boolean isItalic) {

        if (row < 0 || row >= rows || col < 0 || col >= cols) return;

        StyleFlyweight style =
                StyleFactory.getStyle(fontName, fontSize, isBold, isItalic);

        sheet.get(row).set(col, new Cell(text, style));
    }

    // ---------- Get Entry ----------
    public String getEntry(int row, int col) {
        if (row < 0 || row >= rows || col < 0 || col >= cols) return "";

        Cell cell = sheet.get(row).get(col);
        if (cell == null) return "";

        StringBuilder sb = new StringBuilder();
        sb.append(cell.text)
          .append("-")
          .append(cell.style.fontName)
          .append("-")
          .append(cell.style.fontSize);

        if (cell.style.isBold) sb.append("-b");
        if (cell.style.isItalic) sb.append("-i");

        return sb.toString();
    }
}

    üß™ Example

    addEntry(0, 0, "hello", "tahoma", 24, true, false);

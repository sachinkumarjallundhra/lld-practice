    32. Design a Train Platform Management System

    üìú Problem Statement

    Design a system to assign trains to platforms in a railway station while supporting time-based queries.

    Core Requirements

    • At any time, only one train can occupy a platform
    • Platforms are numbered 0 to (platformCount ‚àí 1)
    • Trains may need to wait (delay) if no platform is immediately available
    • Assignment must be deterministic and optimal

    üß† Rules & Constraints

    Platform Rules

    • 1 ‚â§ platformCount ‚â§ 20
    • Arrival and departure times are inclusive
    • If a train departs at time t, the platform becomes free at t + 1

    Delay Rules

    • If no platform is free at arrival time:

      • Train must wait
      • Delay = earliest free time ‚àí arrivalTime

      • Choose the platform with:

      • Minimum delay

      • Lowest platform index in case of tie

    üß© APIs to Implement

    1Ô∏è‚É£ Constructor

    TrainPlatformManager(int platformCount)

    2Ô∏è‚É£ assignPlatform

    String assignPlatform(String trainId, int arrivalTime, int waitTime)
    • Returns "platformIndex,delayTime"
    • Delay is 0 if train can park immediately
    • departureTime = arrivalTime + waitTime - 1 + delay

    3Ô∏è‚É£ getTrainAtPlatform

    String getTrainAtPlatform(int platformNumber, int timestamp)
    • Returns occupying trainId
    • Returns "" if platform is empty at that time

    4Ô∏è‚É£ getPlatformOfTrain

    int getPlatformOfTrain(String trainId, int timestamp)
    • Returns platform index
    • Returns -1 if train is not on any platform at that time

    üìò Example Walkthrough

    TrainPlatformManager mgr = new TrainPlatformManager(3);

// Platforms: 0, 1, 2

mgr.assignPlatform("T1", 0, 5);  // "0,0"  occupies [0..4]
mgr.assignPlatform("T2", 2, 3);  // "1,0"  occupies [2..4]
mgr.assignPlatform("T3", 4, 4);  // "2,0"  occupies [4..7]

mgr.assignPlatform("T4", 5, 5);  // "0,0"  occupies [5..9]

mgr.assignPlatform("T5", 9, 1);  // "1,0"  occupies [9..9]
mgr.assignPlatform("T6", 9, 2);  // "2,0"  occupies [9..10]

mgr.assignPlatform("T7", 9, 3);  // "0,1"  delayed, occupies [10..12]

    üîç Query Examples

    mgr.getTrainAtPlatform(0, 4);   // "T1"
mgr.getTrainAtPlatform(0, 5);   // "T4"
mgr.getTrainAtPlatform(1, 9);   // "T5"
mgr.getTrainAtPlatform(2, 10);  // "T6"

mgr.getPlatformOfTrain("T7", 9);   // -1
mgr.getPlatformOfTrain("T7", 10);  // 0
mgr.getPlatformOfTrain("T6", 11);  // -1

    üèóÔ∏è Design Overview

    Core Entities

    Entity Responsibility
    Platform Holds scheduled intervals
    Interval Represents train occupancy
    TrainRecord Fast train-centric lookup
    TrainPlatformManager Orchestrates assignment & queries

    üß† Design Principles Used

    • Greedy scheduling
    • Interval scanning
    • Deterministic tie-breaking
    • Separation of concerns

    • Easily extensible for:

      • Priority trains
      • Multiple stations
      • Platform maintenance windows

    üíª Java Implementation

    import java.util.*;

public class TrainPlatformManager {

    static class Interval {
        String trainId;
        int start;
        int end;

        Interval(String trainId, int start, int end) {
            this.trainId = trainId;
            this.start = start;
            this.end = end;
        }
    }

    static class Platform {
        int index;
        List<Interval> schedule = new ArrayList<>();

        Platform(int index) {
            this.index = index;
        }
    }

    static class TrainRecord {
        String trainId;
        int platform;
        int start;
        int end;

        TrainRecord(String trainId, int platform, int start, int end) {
            this.trainId = trainId;
            this.platform = platform;
            this.start = start;
            this.end = end;
        }
    }

    private Platform[] platforms;
    private Map<String, TrainRecord> trainMap = new HashMap<>();

    public TrainPlatformManager(int platformCount) {
        platforms = new Platform[platformCount];
        for (int i = 0; i < platformCount; i++) {
            platforms[i] = new Platform(i);
        }
    }

    public String assignPlatform(String trainId, int arrivalTime, int waitTime) {

        int bestPlatform = -1;
        int bestDelay = Integer.MAX_VALUE;
        int bestStart = -1;

        for (Platform p : platforms) {

            int start = earliestStart(p, arrivalTime, waitTime);
            int delay = start - arrivalTime;

            if (delay < bestDelay ||
               (delay == bestDelay && p.index < bestPlatform)) {

                bestDelay = delay;
                bestPlatform = p.index;
                bestStart = start;
            }
        }

        int end = bestStart + waitTime - 1;

        Interval interval = new Interval(trainId, bestStart, end);
        platforms[bestPlatform].schedule.add(interval);

        platforms[bestPlatform].schedule.sort(
            (a, b) -> Integer.compare(a.start, b.start)
        );

        trainMap.put(trainId,
            new TrainRecord(trainId, bestPlatform, bestStart, end));

        return bestPlatform + "," + bestDelay;
    }

    public String getTrainAtPlatform(int platformNumber, int timestamp) {
        Platform p = platforms[platformNumber];

        for (Interval in : p.schedule) {
            if (in.start <= timestamp && timestamp <= in.end) {
                return in.trainId;
            }
        }
        return "";
    }

    public int getPlatformOfTrain(String trainId, int timestamp) {
        TrainRecord tr = trainMap.get(trainId);
        if (tr == null) return -1;

        if (tr.start <= timestamp && timestamp <= tr.end) {
            return tr.platform;
        }
        return -1;
    }

    private int earliestStart(Platform p, int arrivalTime, int waitTime) {

        int start = arrivalTime;

        for (Interval in : p.schedule) {
            int end = start + waitTime - 1;

            if (end < in.start) {
                return start;
            }

            if (start <= in.end) {
                start = in.end + 1;
            }
        }

        return start;
    }
}

    ‚ö†Ô∏è Edge Cases Covered

    • Immediate reuse after departure
    • Multiple platforms freeing at same time
    • Delayed scheduling
    • Exact boundary overlaps
    • Train not yet started / already departed

    Where:
